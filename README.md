# Bank Churn Analysis 

This project focuses on analyzing customer retention in a banking dataset using **Microsoft Excel**. The goal was to identify patterns and factors that influence whether a customer stays with or leaves the bank.
##  Data Process
1.  **Data Cleaning:** Used Excel (Power Query) to handle missing values and standardize columns.
2.  **Exploratory Data Analysis (EDA):** Created Pivot Tables to compare churn across different demographics (Geography, Gender, Age).
3.  **Visualization:** Built charts to highlight the "at-risk" customer segments.
